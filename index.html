Built with ease.

The following elements, namely HTML, CSS and Bootstrap combined together render the website functional. These parts are integral in the frontend design of the project and make possible the manifestation of the website on your screen.

HTML5

Html stands for Hyper Text Markup Language. It forms the basic structure of the website. It provides tags that define the complete webiste.

BOOTSTRAP

Bootstrap is an open source frontend framework which provides templates for various HTML and CSS components which are free to use in your website.

CCS3

CSS stands for Cascading Style Sheets. It provides the frontend styling by describing how elements should be displayed.

Our Philosophy

Idea Visulo was created with the belief that creativity is fundamental for growth. Idea Visulo is committed to make complex math and physics concepts easier to grasp and fun to learn by creative visualisation. It aims to encourage greater interest in concepts by offering a look into how things actually work and helping people look at the same theory in a new light.
